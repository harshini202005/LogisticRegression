# 🚀 Logistic Regression Model for Bank Personal Loan Prediction

This project builds a **Logistic Regression** model using a dataset of personal and financial attributes to predict whether a customer will take a personal loan. It also includes visualizations and model evaluation metrics.

---

## 📊 Dataset

- **Filename**: `Bank_Personal_Loan_Modelling.csv`
- **Target Variable**: `Personal Loan` (0 = No, 1 = Yes)
- **Features Used**:
  - ID
  - Age
  - Experience
  - Income
  - CCAvg
  - Education
  - Mortgage
  - CD Account
  - CreditCard

---

## 🛠 Technologies Used

- Python
- Pandas, NumPy
- Matplotlib, Seaborn (for visualization)
- Scikit-learn (Logistic Regression, Model Evaluation)
- Pickle (to save the model)

---

## 🧠 Model Workflow

1. Load and visualize data (Age vs Income bar chart).
2. Select features and handle missing values.
3. Split dataset into training and testing sets.
4. Train a Logistic Regression model.
5. Evaluate the model using:
   - Accuracy
   - Confusion Matrix
   - Classification Report
6. Visualize correlation heatmap.
7. Save the trained model as `model.pkl`.
